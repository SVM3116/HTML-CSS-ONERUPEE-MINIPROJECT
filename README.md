# Day 11-08.01.2026: One Rupee Photo Gallery

This project is a simple photo gallery website titled "One Rupee Photo," featuring an animated sidebar menu. It demonstrates CSS animations, transitions, and responsive design for a sidebar navigation.

## Files

- **photo.html**: The main HTML page for the photo gallery. It includes:
  - A hamburger menu button to toggle the sidebar.
  - A sidebar with navigation links (Gallery, Shortcuts, Exhibits, Events, Store, Contact, Feedback).
  - Social media icons at the bottom of the sidebar.
  - Background image set via CSS.

- **styles.css**: The CSS stylesheet that styles the page and handles the sidebar animations. Key features include:
  - Sidebar that slides in from the left when the menu button is checked.
  - Smooth transitions for menu icons and hover effects.
  - Background image covering the full viewport.
  - Font styling using Poppins and other Google Fonts.
  - Responsive behavior with opacity changes and scaling on hover.

## Technologies Used

- HTML5
- CSS3 (animations, transitions, flexbox-like layouts via positioning)

## How to View

1. Open `photo.html` in a web browser.
2. Click the hamburger menu icon to toggle the sidebar.
3. Hover over menu items and social icons to see animations.
